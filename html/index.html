HTML Tutorial

Welcome to our HTML tutorial! Here, you'll learn about different HTML tags and their usage with examples.

<!DOCTYPE html> - Document Type Declaration

The DOCTYPE declaration is the first line of an HTML document, specifying the HTML version being used.

Example:

<!DOCTYPE html>

This declares that the document is using HTML5.

<html> - HTML Root Element

The html tag is the root element of an HTML page, containing all other elements.

Example:

<html lang="en">
  <!-- Other HTML content goes here -->
</html>

The lang attribute specifies the language of the document.

<head> - Document Metadata

The head tag contains metadata about the HTML document, such as title, character encoding, and linked resources.

Example:

<head>
  <title>My Web Page</title>
  <!-- Other metadata goes here -->
</head>

<meta> - Metadata Tags

Meta tags provide additional information about the document, often used for SEO and proper rendering.

Examples:

<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="description" content="A brief description of the page content">

These meta tags set the character encoding, adjust viewport for responsive design, and provide a page description, respectively.

<body> - Document Body

The body tag contains the visible content of an HTML document, including text, images, and other elements that are displayed in the browser.

Example:

<body>
  <h1>Welcome to My Website</h1>
  <p>This is the main content of the page.</p>
</body>

Everything inside the body tag is what you see on the webpage. It's where you put all the content that you want to show to your visitors.

<h1> to <h6> - Heading Tags

Heading tags are used to define different levels of headings in HTML, with h1 being the highest level and h6 the lowest.

Example:

<h1>Main Heading</h1>
<h2>Subheading</h2>
<h3>Smaller Subheading</h3>

Output:

Main Heading

Subheading

Smaller Subheading

<div> - Division Tag

The div tag is a container used to group other HTML elements for styling purposes.

Example:

<div class="container">
  <p>This is inside a div.</p>
</div>

<span> - Inline Container Tag

The span tag is an inline container used to mark up a part of a text, or a part of a document.

Example:

<p>This is <span style="color: red;">red</span> text.</p>

<p> - Paragraph Tag

The paragraph tag is used to define a paragraph of text.

Example:

<p>This is a paragraph of text.</p>

Result:

This is a paragraph of text.

<a> - Anchor Tag

The anchor tag is used to create hyperlinks to other web pages or resources.

Example:

<a href="https://www.example.com">Visit Example.com</a>

Result:

Visit Example.com

<img> - Image Tag

The image tag is used to embed images in your HTML document.

Example:

<img src="image.jpg" alt="Description of the image">

Result: (Replace 'image.jpg' with an actual image URL to see the result)

<ul> and <li> - Unordered List and List Item Tags

These tags are used to create bulleted lists.

Example:

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
  </ul>

Result:

  • Item 1
  • Item 2
  • Item 3

These are just a few examples of HTML tags. Continue exploring to learn more about HTML and web development!

<table>, <tr>, <th>, and <td> - Table Tags

These tags are used to create tables in HTML.

Example:

<table>
  <tr>
    <th>Header 1</th>
    <th>Header 2</th>
  </tr>
  <tr>
    <td>Row 1, Cell 1</td>
    <td>Row 1, Cell 2</td>
  </tr>
</table>

Result:

Header 1 Header 2
Row 1, Cell 1 Row 1, Cell 2

The div tag is used as a container for other HTML elements and for styling purposes.

Example:

<div style="background-color: #f0f0f0; padding: 10px;">
